Meu Manifesto:

  • Um bom design é menos: menos é mais
  • Menos é Ousado: Ousado significa confiança
  • Ousado é experimental: experimentos envolvem riscos
  • O design experimental é novo: rostos novos são sempre bem-vindos
  • Fresh é criativo: a criatividade atrai a curiosidade
  • Criativo é inteligente: faça as pessoas pensarem e repensarem
  • Inteligente é comunicativo: comunique-se visualmente
  • O design comunicativo é simples: fácil de entender, mas difícil de produzir
  • Simples é estético: coisas atraentes funcionam melhor
  • A estética é boa: bom o suficiente nunca é bom o suficiente
